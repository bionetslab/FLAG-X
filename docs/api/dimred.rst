
flagx.dimred
============

The ``flagx.dimred`` module exposes selected dimensionality reduction classes from **umap-learn** and **scikit-learn**.

For documentation, refer to the respective upstream libraries.

Available classes:

- ``UMAP`` (umap-learn)
- ``PCA`` (scikit-learn)
- ``MDS`` (scikit-learn)
- ``TSNE`` (scikit-learn)
- ``LocallyLinearEmbedding`` (scikit-learn)
- ``Isomap`` (scikit-learn)
- ``SpectralEmbedding`` (scikit-learn)
